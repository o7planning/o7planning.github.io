BlockQuickItemUpdateAction

Thay vì sử dụng Form để tạo và cập nhập các ITEM của Block bạn có thể sử dụng BlockQuickItemCreationAction, BlockQuickItemUpdateAction.

Mục tiêu của ví dụ:
  • Sử dụng BlockQuickItemCreationAction để cập nhập trạng thái VIP của một Customer.
  • Sử dụng BlockQuickItemUpdateAction để cập nhập code của một Customer.
